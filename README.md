# Coherence Plugin

Unified toolkit for Atomization, Summarization, and Wisdom Extraction in Obsidian.

## Features

1.  **Atomizer**: Split large markdown files by Heading, ISO Date, or Divider.
2.  **Summarizer**: AI-powered summarization of notes and folders using Ollama.
3.  **Wisdom Extractor**: Rewrite personal notes into general wisdom using AI (Ollama).

## Installation

### From Community Plugins (Coming Soon)
1.  Open **Settings** > **Community Plugins**.
2.  Turn on **Restricted mode** (if off).
3.  Click **Browse**.
4.  Search for **Coherence Wizard**.
5.  Click **Install** and then **Enable**.

### Manual Installation
1.  Download the latest release from the [Releases](https://github.com/Coherence-Wizard/Coherence-Wizard/releases) page.
2.  Extract the files into your vault's `.obsidian/plugins/obsidian-coherence-plugin/` folder.
3.  Reload Obsidian.
4.  Enable **Coherence Wizard** in **Community Plugins**.

## Usage

*   **Atomizer**: Open Command Palette (`Ctrl/Cmd + P`) and search for `Coherence: Open Atomizer`.
*   **Summarizer**: Open Command Palette and search for `Coherence: Open Summarizer`.
*   **Wisdom Extractor**: Open Command Palette and search for `Coherence: Open Wisdom Extractor`.

## Configuration

Go to **Settings > Coherence Plugin** to configure your Ollama URL and default model.
